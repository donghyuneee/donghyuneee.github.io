## 변수 값 변경
처음 변수 a에 값을 넣고 이후 a에 다른 값을 넣으면
이후에 넣었던 값이 최종적으로 담겨있게 된다.

```java
public class Main {
    public static void main (String args[]) {
        int a = 5;
        System.out.println("A is " + a);

        a = 3;
        System.out.println("A is " + a);
    }
}
```
